# PROJETO-TRILOGIA-DO-SABER-SENAI
Desenvolver um sistema educacional em Python, fora de ambiente web, que avalie o nível de conhecimento dos alunos do SENAI, identifique lacunas de aprendizagem e direcione estudos práticos ligados ao mundo do trabalho, melhorando o desempenho no SAEP e nas avaliações internas.
