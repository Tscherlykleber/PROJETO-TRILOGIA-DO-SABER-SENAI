# PROJETO-TRILOGIA-DO-SABER-SENAI
📘 1. PROJETO EDUCACIONAL – SENAI
Nome do Projeto
EDUCA TRABALHO SENAI – Sistema Inteligente de Apoio ao SAEP
Objetivo Geral
Desenvolver um sistema educacional em Python, fora de ambiente web, que avalie o nível de conhecimento dos alunos do SENAI, identifique lacunas de aprendizagem e direcione estudos práticos ligados ao mundo do trabalho, melhorando o desempenho no SAEP e nas avaliações internas.
Disciplinas Atendidas
✅ Matemática
✅ Informática
✅ Língua Portuguesa
Público-Alvo
Alunos dos cursos técnicos e de aprendizagem industrial do SENAI.                                                         🎯 2. FUNCIONALIDADES DO SISTEMA
✔ Cadastro do aluno
✔ Avaliação diagnóstica por disciplina
✔ Pontuação por área
✔ Classificação de nível:

Básico
Intermediário
Avançado

✔ Recomendações de estudo ligadas ao mundo do trabalho
✔ Relatório final no próprio terminal
✔ Estrutura preparada para evolução (PDF, banco de dados, interface gráfica)

🧠 3. LÓGICA PEDAGÓGICA (ALINHADA AO SAEP)
O sistema funciona assim:

O aluno responde questões contextualizadas no mundo do trabalho
O sistema calcula o desempenho por disciplina
Identifica pontos fracos
Recomenda conteúdos práticos, como:

Matemática aplicada à indústria
Informática para ambiente corporativo
Português profissional (e-mails, relatórios, interpretação)



Modelo do SAEP, que avalia competência e aplicação prática, não só teoria.

🖥️ 4. SISTEMA EM PYTHON (FORA DA WEB)
✅ Funciona no terminal
✅ Apenas Python padrão
✅ Pode rodar no Windows, Linux ou Mac
